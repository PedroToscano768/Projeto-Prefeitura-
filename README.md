O Prefeitura+ tem como propósito fortalecer a comunicação entre cidadãos e administração pública, oferecendo um meio rápido e acessível para registrar problemas do cotidiano urbano. A plataforma funciona como um canal direto de comunicação, permitindo que moradores relatem situações que muitas vezes passam despercebidas pelo poder público, contribuindo para ações mais eficientes e para a melhoria contínua da cidade.

No desenvolvimento do sistema, utilizamos TypeScript como linguagem principal, além de um pouco de JavaScript. O projeto também faz uso de um banco de dados SQL e das bibliotecas Jest, bcrypt e JWT para testes, segurança e autenticação.
